1.安装脚手架 cnpm install -g vue-cli 全局下安装脚手架  这个就是有了可以在控制台输入命令的东西了
2.创建一个基于webpack的项目  cnpm init webpack renlun
3.切换到cd  你的项目名字
4.安装依赖  cnpm  install
